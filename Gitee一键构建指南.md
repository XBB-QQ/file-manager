# 🚀 Gitee 一键构建APK指南（国内访问更快！）

## 📱 纯手机操作，5分钟搞定！

---

## 第一步：创建Gitee仓库

1. 手机浏览器打开 **https://gitee.com**
2. 注册/登录账号（用手机号即可）
3. 点击右上角 **"+"** → **"新建仓库"**
4. 填写：
   - 仓库名称：`文件管理器`
   - 选择 **公开**（免费）
   - ✅ 勾选 "使用 README.md 初始化"
5. 点击 **"创建"**

---

## 第二步：上传项目文件

1. 在仓库页面点击 **"上传文件"**
2. 选择 `/workspace` 文件夹里的所有文件
3. 可以先压缩成 zip 再上传
4. 点击 **"提交"**

---

## 第三步：开启 Gitee Actions

1. 在仓库页面点击 **"CI/CD"** → **"Actions"**
2. 如果提示未开启，点击 **"开启 Actions"**
3. 会自动检测到 `build-apk.yml` 配置文件
4. 点击 **"运行工作流"** 或等待代码推送后自动运行

---

## 第四步：下载APK

1. 等待 3-5 分钟构建完成
2. 在 Actions 页面点击构建记录
3. 下载构建产物（app-debug.apk）
4. 安装到手机！

---

您的文件管理器APK已经到手！🎉

## 📁 重要文件

- `.gitee/workflows/build-apk.yml` - Gitee Actions 配置
- `.github/workflows/build-apk.yml` - GitHub Actions 配置
- `android/` - Android 项目
- `dist/` - Web 应用构建产物

## ⚠️ 注意事项

1. Gitee 需要实名认证（国内平台要求）
2. 确保仓库是公开的（私有仓库 Actions 可能有限制）
3. 如果第一次构建失败，点击重试即可
