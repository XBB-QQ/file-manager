# 📱 纯手机操作 - 5分钟构建APK指南

## 只需要手机！不用电脑！

---

## 第一步：创建GitHub仓库（手机操作）

1. 用手机浏览器打开 **https://github.com**
2. 登录或注册GitHub账号（用手机号就可以注册）
3. 点击右上角 **"+"** → **"New repository"**
4. 填写：
   - Repository name: `文件管理器` 或任意名称
   - 选择 **Public**（免费构建）
   - ✅ 勾选 "Add a README file"
5. 点击 **"Create repository"**

---

## 第二步：上传项目文件

1. 在刚创建的仓库页面
2. 点击 **" uploading an existing file"**（或者点击 "Add file" → "Upload files"）
3. 在电脑上打开 `/workspace` 文件夹，把所有文件拖到上传区域
4. 或者用手机连接电脑，把文件传到手机，再用手机浏览器上传
5. 点击 **"Commit changes"**

---

## 第三步：等待自动构建

1. 推送完成后，点击仓库的 **"Actions"** 标签
2. 会看到 "Build APK" 工作流正在运行
3. 等待 **3-5分钟** 构建完成

---

## 第四步：下载APK

1. 构建完成后，点击该构建
2. 点击 **"app-debug"** 
3. 点击下载按钮
4. APK下载完成！

---

## 第五步：安装到手机

1. 打开下载的 APK 文件
2. 允许安装（如果提示未知来源，进入设置允许）
3. 安装完成！

---

## ✅ 完成！

您的文件管理器APK已经可以使用了！🎉

---

## 💡 小提示

- 如果第二步上传文件困难，可以先把 `/workspace` 压缩成 zip 文件再上传
- GitHub Actions 是免费的，每天可以构建多次
- 构建的APK是调试版本，可以直接安装在手机上测试
