# 文件管理器 - 手机部署指南

## 📱 方法一：本地电脑运行（推荐）

### 1. 将项目复制到您的电脑
   复制整个 `/workspace` 文件夹到您的电脑

### 2. 安装依赖并运行
```bash
cd /path/to/workspace
npm install
npm run dev
```

### 3. 手机访问
   查看您电脑的局域网IP（Windows: `ipconfig`, Mac/Linux: `ifconfig`）
   在手机浏览器访问：`http://您电脑的IP:5173/`

---

## 🌐 方法二：部署到免费托管

### 使用 Vercel（最简单）
1. 访问 https://vercel.com
2. 注册/登录
3. 导入项目
4. 部署！

### 使用 Netlify
1. 访问 https://netlify.com
2. 直接将 `dist/` 文件夹拖放到网站上
3. 立即获得公网地址！
